# MCP Currency Converter

A custom MCP Server built using Python that integrates with Claude Desktop and performs real-time currency conversion.

## Features

- Claude Desktop Integration
- MCP Tool Calling
- Real-time Currency Conversion
- Python Implementation

## Architecture

Claude Desktop
      ↓
MCP Server
      ↓
Currency API
      ↓
Response


## Demo

This MCP Server integrates with Claude Desktop and provides real-time currency conversion.

Example Prompt:
Convert 100 USD to INR

Example Response:
100 USD = 8,350 INR (example value)

## Tech Stack

- Python
- MCP (Model Context Protocol)
- Claude Desktop
- Exchange Rate API
